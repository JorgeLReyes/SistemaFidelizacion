# Configuración del proyecto

## Requisitos

Antes de ejecutar la aplicación es necesario:

- Tener una cuenta de **Firebase**.
- Crear un proyecto en Firebase.
- Habilitar una base de datos (Firestore).
- Registrar una aplicación **Web** dentro del proyecto de Firebase.

## Variables de entorno

Crea un archivo `.env` en la raíz del proyecto con las siguientes variables:

```env
VITE_FIREBASE_API_KEY=""
VITE_FIREBASE_AUTH_DOMAIN=""
VITE_FIREBASE_PROJECT_ID=""
VITE_FIREBASE_STORAGE_BUCKET=""
VITE_FIREBASE_MESSAGING_SENDER_ID=""
VITE_FIREBASE_APP_ID=""
```

Estas credenciales se obtienen desde la configuración de la aplicación Web en Firebase.

## Pasos de configuración

1. Crear un proyecto en Firebase.
2. Registrar una aplicación **Web**.
3. Crear y configurar la base de datos **Cloud Firestore**.
4. Copiar las credenciales de Firebase al archivo `.env`.
5. Desplegar el proyecto en la plataforma de tu preferencia (por ejemplo, Vercel).

Una vez configuradas las variables de entorno, la aplicación estará lista para conectarse a Firebase.
